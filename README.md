## Run Locally

Clone the project

```bash
  git clone https://github.com/sridhar02/sanity-landing-page-studio
```

Go to the project directory

```bash
  cd sanity-landing-page-studio
```

Install dependencies

```bash
  yarn
```

Start the server

```bash
  sanity start
```
